## Secret Santa

### Clone project
```bash
$ git clone https://github.com/YarikHaidai/secret-santa.git secret-santa
$ cd secret-santa
```

### Installation
```bash
$ npm install
```
### Configure local environment
```bash
$ cp .env.example .env
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Swagger Documentation
```
http://localhost:3000/api
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
